# 🤖 Любава — AI-персонаж для спілкування

Віртуальна супутниця з системою рівнів, голосовими повідомленнями та монетизацією.

## 🚀 Технології

- Python 3.11+
- Aiogram 3.x — асинхронний фреймворк для Telegram
- Groq Llama 3.3-70B — генерація тексту (500 токенівсек)
- ElevenLabs — голосові повідомлення
- SQLite — зберігання даних (WAL режим)
- Telegram Stars — монетизація
